# 미니 방치형 정원 게임

기획안 기반으로 만든 정적 웹 MVP입니다. 별도 설치 없이 `index.html`을 브라우저로 열면 바로 실행됩니다.

## 실행

1. `index.html`을 더블클릭해서 브라우저로 엽니다.
2. 씨앗을 선택하고 빈 밭을 눌러 심습니다.
3. 작물이 다 자라면 밭을 눌러 수확합니다.
4. 인벤토리에서 판매하거나 오늘의 손님 요청에 납품합니다.

## 구현된 기능

- 3×3 밭, 시작 4칸 개방, 골드로 밭 확장
- 씨앗 구매, 심기, 시간 경과 성장, 수확
- 일반/은빛/황금/시든 품질
- localStorage 자동 저장과 재접속 성장 계산
- 황금 물뿌리개 일일 보상
- 숲 입구 채집 포인트와 리필
- 오늘의 손님 요청
- 인벤토리 판매, 모두 판매
- 도감 등록과 달성 보상
- PC/모바일 반응형 UI

## 밸런스 메모

현재 값은 플레이 테스트가 쉬운 데모 밸런스입니다.

- 작물 성장: 45초~10분
- 채집 리필: 5분
- 시듦: 완성 후 45분
- 오프라인 성장 계산 상한: 12시간

실서비스 리듬으로 바꾸려면 `game.js` 상단의 `CROP_DEFS`, `GATHER_REFILL_MS`, `WILT_AFTER_MS` 값을 조정하면 됩니다.
